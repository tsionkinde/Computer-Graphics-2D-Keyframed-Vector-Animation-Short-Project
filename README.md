## 2D Keyframed Vector Animation

A short computer graphics project that demonstrates the principles of 2D vector-based animation using keyframing techniques.
The project focuses on smooth motion interpolation, object transformation, and scene animation using computer graphics concepts.

## Features
2D vector object rendering
Keyframe-based animation system
Smooth interpolation between frames
Translation, rotation, and scaling transformations
Multiple animated objects
Scene timeline control
Lightweight and easy-to-understand implementation
Project Overview

This project was developed as part of a Computer Graphics course to demonstrate how animations are created using keyframes.

In traditional animation, artists draw many frames manually. In computer graphics, we can define only important frames called keyframes, and the computer automatically generates the intermediate frames through interpolation.

The animation system in this project allows:

Defining object states at specific times
Interpolating motion smoothly
Animating vector-based shapes efficiently
Technologies Used
Programming Language:  Python 
Graphics Library: OpenGL
